# Potent Engine
A sort of game engine, quite bare bone right now

## Features:
- Mass agregate physics
- Deferred rendering
- Deferred light shaders
- Component system
- Scene system
- Assets (unfinished for now)
- Collision detection

## Libraries used:
- GLFW - for window handling
- GLAD - for opengl context
- stb - for loading images
- mono - for C# scripts
- lua - for Lua scripts
